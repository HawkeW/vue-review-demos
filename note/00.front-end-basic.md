#### margin纵向重叠的问题

- 相邻元素的margin-top和margin-bottom会发生重叠

- 空白内容的`<p></p>`也会重叠

```html
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
  <style>
    p {
        font-size: 16px;
        line-height: 1;
        margin-top: 10px;
        margin-bottom: 15px;
    }
</style>
</head>
<body>
  <div id="div1">
   <p>AAA</p>
   <p></p>
   <p></p>
   <p></p>
   <p>BBB</p>
  </div>
</body>
</html>
```
